# kynetic-renderer

This is the renderer for use with JSON documents generated by the cloud-based editor.

## Features

- Convert JSON project definitions into rendered video with `handanim`
- Supports standard drawables: `Text`, `Square`, `Rectangle`, `Line`, `Polygon`, `SVG`.
- Supports basic animations: `TranslateTo`, `FadeIn`, `FadeOut`, `ZoomOut`, `Sketch`.
- Dockerized Python rendering pipeline.

## How to use

### Docker (recommended)
Coming soon!

### Locally via `poetry`

Install dependencies:
```bash
poetry install
```

Render a project:
```bash
poetry run kynetic-render path/to/project.json
```
